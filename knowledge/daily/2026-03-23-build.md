## Technische Details
- Horizontal Scroll: `gsap.to(track, { x: -totalWidth, scrub: 1, pin: true })`
- Stacking Cards: `position: sticky; top: 5vh;` mit steigendem z-index + paddingTop
- Counter Stats: `gsap.to(obj, { val: target, onUpdate: ... })` mit toggleActions
- Preloader: Einfacher useState-Timer mit CSS opacity-Transition
- ScrollTrigger.matchMedia ersetzt durch gsap.matchMedia() (TS type fix)

- **CSS sticky Stacking Cards** — kein JavaScript noetig, funktioniert zuverlaessig
- **gsap.matchMedia()** — responsives Verhalten fuer ScrollTrigger-basierte Layouts
- **Mehr Content = weniger langweilig** — About-Section + expanded descriptions machen riesen Unterschied
- **Feedback-Driven Rebuild > Polish** — fundamental neubauen bringt mehr als Kleinigkeiten polishen
