# Brief: hero-v001-b

## Erlebnis-Kategorie
Typografie-Only — Nur Schrift, null Bilder, pure Kinetik

## Branche
Architekturbüro — "FORM STEIN RAUM" — Minimal, brutalistische Architektur

## Persona
Experimenteller Typograf — Buchstaben als Raum, Text als Skulptur

## Aufgabe
Baue einen Hero-Abschnitt der Typografie als interaktives Erlebnis behandelt. Kein Bild. Die Buchstaben SIND die Architektur. Experimentiere mit Masken, Clipping, Mix-Blend-Modes, CSS Grid für unkonventionelle Text-Platzierung. Der Text soll sich anfühlen wie eine begehbare Installation.

## Skills
- motion-framer (AnimatePresence, layout animations, gestures)
- creative-effects (mix-blend-mode, noise overlay, grain)

## Constraints
- Viewport-füllend (100vh)
- Monochrom (Schwarz + Off-White #f5f0eb)
- Keine Bilder, keine Icons
- Text: "FORM STEIN RAUM" + "Architektur die bleibt"
- Mobile-first (390px), dann Desktop
